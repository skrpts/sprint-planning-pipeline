## Capability

Decomposes high-level epics and feature requests into well-scoped user stories sized for a single sprint.

## When to Use

- Sprint planning ceremonies
- Backlog refinement sessions
- Breaking down new feature requests

## Inputs

Epic description, team velocity, sprint duration

## Outputs

User stories with acceptance criteria, story points, and dependency notes
